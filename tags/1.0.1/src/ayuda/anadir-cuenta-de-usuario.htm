Añadir cuenta de usuario
 Previous Top Next
La opción de añadir una cuenta ya existente está disponible en el gestor de las cuentas del sistema. Para acceder a ella es necesario seleccionar la opción "Añadir cuenta", que está disponible en dicho gestor.


Una vez seleccionada la opción, se presenta un formulario al usuario en el que puede escoger entre crear una cuenta nueva en un servidor o añadir una cuenta ya existe. En nuestro caso se debe seleccionar la segunda opción.


Entonces el sistema muestra un formulario, en el que el usuario debe introducir los datos para añadir una cuenta ya existente.


Después de introducir los datos correctamente y seleccionar la opción "Aceptar", el gestor de cuentas añade la nueva cuenta a la lista de cuentas.
