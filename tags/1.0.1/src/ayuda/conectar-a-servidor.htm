Conectar a servidor
 Previous Top Next
Se habilitan dos puntos en los que el usuario puede invocar la operación, el primero se encuentra en el menú Acciones del sistema y el segundo en la barra de herramientas de la ventana principal. Ambas opciones sólo estarán disponibles si el usuario todavía no está conectado.


Si se selecciona cualquiera de las alternativas, la conexión se realizará con la cuenta activa en este instante, y si todo transcurre normalmente el sistema mostrará la lista de contactos del usuario.


Si el sistema no puede recuperar la contraseña de la cuenta para realizar la conexión, muestra un formulario en el que el usuario puede introducir la contraseña de la cuenta.


Una vez introducida la contraseña, el sistema se comporta como en el caso anterior.
