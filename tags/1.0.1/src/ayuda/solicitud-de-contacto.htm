Solicitud de contacto

Se habilitan dos puntos en los que el usuario puede invocar la operación, el primero se encuentra en el menú Acciones del sistema con el nombre de "Añadir contactos" y el segundo en la barra de herramientas de la ventana principal.


Cuando se selecciona cualquiera de las opciones el sistema muestra un formulario en el que el usuario debe introducir los datos del contacto que desea establecer.
