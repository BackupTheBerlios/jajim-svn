Rechazar invitación
 Previous Top Next
El sistema muestra una ventana popup en la que informa de que una invitación a un chat multiusuario ha sido recibida.


Cuando el usuario selecciona la ventana popup, se muestra la ventana de la aplicación junto con un formulario en el que el usuario puede aceptar o rechazar la invitación.


Si el usuario selecciona el botón ·"Rechazar", el sistema envía una notificación con el rechazo de la invitación al contacto que la envió. Además, si el usuario deja que se minimice la ventana ventana popup sin seleccionarla, el sistema rechaza la petición por defecto.
