Ban contact in multiuser chat
 Previous Top Next
Option "Ban contact" is only available in multiuser chat. There are two place in which user can select this Actions, in "Actions" menu and toolbar of multiuser chat.


When user selects either option, system shows a form in which user can select contact that he wants to ban.


When user selecst one or more contacts and click in "OK" button, system sends a notification to this contacts.
