Accept invitation
 Previous Top Next
System shows a popup window in which it says that a request from multiuser chat has been received.


When user selects this window, system shows application window and a form in which he can accept or reject this invitation.


For accept this invitation user must select "Accept" button. When this event success, system opens a form.


In this form user puts into his alias. After, he selects option "OK", and system opens a new conversation window.
