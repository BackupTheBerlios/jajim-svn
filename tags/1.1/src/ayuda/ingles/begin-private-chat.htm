Create multiuser chat
 Previous Top Next
A multiuser chat is a conversation in which two or more participants can talk. Invite to other participants is possible.

For create a multiuser chat, you must go to contacts panel and select contact that you want to talk. This action open a popup menu.


When user selecst option "Multiuser chat", system shows a form in which user must put room name and nick that he will use in the chat.


When user fills fields of form and select option "OK", system creates a room, opens a multiuser chat window and sends a invitation to selected contact.


In this window options "Invite contact" and "Ban contact" are availables.
