Change account
 Previous Top Next
With this option, user can select a account that replace the current main account. User only can connect to server with main account. Main account is showed in main window of application.


"Change account" option is available in account manager.



For change main account, user must select other account in account list and click in "Activate account" option. System aborts connection, if user was connected with main account. Main window is updated with new main account.
