Reject invitation
 Previous Top Next
System shows a popup window in which it says that a multiuser chat request has been received.


When user selects this popup window, system shows main window and a form in which user can accept or reject this invitation.


For reject this invitation user must select "Reject" button. When this event success, system sends a message that it confirms to contact this rejected.

Also, if user doesn't select popup window, system rejects request by default.
