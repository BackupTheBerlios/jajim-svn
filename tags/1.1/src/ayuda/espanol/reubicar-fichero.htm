Reubicar fichero
 Previous Top 
Esta opción se encuentra disponible en la ventana del gestor de transferencias, concretamente en la parte superior de la pestaña de ficheros descargados.


Cuando el usuario selecciona el fichero que desea reubicar y le da a la opción correspondiente, el sistema muestra el siguiente formulario.


Si el usuario rellena el formulario correctamente y selecciona la opción "Aceptar", el sistema mueve el fichero a la carpeta introducida.
