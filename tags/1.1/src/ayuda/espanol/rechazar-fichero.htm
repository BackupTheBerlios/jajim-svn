Rechazar fichero
 Previous Top Next
El sistema muestra una ventana popup en la que informa de que una petición de transferencia de fichero ha sido recibida.


Cuando el usuario selecciona la ventana popup, se muestra la ventana de la aplicación junto con un formulario en el que el usuario puede aceptar o rechazar la petición de transferencia.


Si el usuario selecciona el botón "Rechazar", el sistema cancela la transferencia y envía una notificación al contacto.

Además, si el usuario no selecciona la ventana popup que contiene la petición de transferencia, el sistema automáticamente rechaza la solicitud de transferencia.
