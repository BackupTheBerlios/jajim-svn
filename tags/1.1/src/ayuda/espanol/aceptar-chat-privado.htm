Aceptar chat privado
 Previous Top Next
El sistema muestra una ventana popup en la que informa de que una petición de chat privado ha sido recibida.


Cuando el usuario selecciona la ventana popup, se muestra la ventana de la aplicación junto con un formulario en el que el usuario puede aceptar o rechazar la petición de chat privado.


Para aceptar la petición el usuario debe seleccionar el botón "Aceptar", en ese instante el sistema abre una ventana de conversación para que converse el usuario con su contacto.
