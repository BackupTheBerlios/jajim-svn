Aceptar invitación
 Previous Top Next
El sistema muestra una ventana popup en la que informa de que una invitación a un chat multiusuario ha sido recibida.


Cuando el usuario selecciona la ventana popup, se muestra la ventana de la aplicación junto con un formulario en el que el usuario puede aceptar o rechazar la invitación.


Cuando el usuario selecciona la opción "Aceptar" el sistema muestra el siguiente formulario.


En el formulario el usuario introduce el alias que va a utilizar durante la conversación. Luego de introducirlo correctamente y selecciona Aceptar, el sistema abre una nueva ventana de conversación.
