Antes de comenzar la instalación, debe comprobar que su sistema tiene instalada la máquina virtual de java. Se recomienda la utilización de la versión 6, o superior de la misma. No se garantiza el correcto funcionamiento de la herramienta en versiones anteriores.

Instalación en Windows

Para iniciar la instalación en Windows, hay que ejecutar el instalador del programa. Durante la ejecución del mismo se ofrece al usuario la posibilidad de seleccionar la ruta de instalación.


Cuando el usuario selecciona la ruta, el instalador comienza a copiar los ficheros necesarios en el sistema. Finalmente, incluye un enlace al programa en el menú Inicio del sistema.


Instalación en Linux

Para instalar el programa en Linux es necesario disponer del fichero JAJIM-<version>.tar.gz. Primero, se debe descomprimir el mismo, después se entra dentro del directorio generado después de la descompresión. Finalmente, se ejecuta como “root “ el script instalador.sh.

Cuando finaliza el proceso de instalación, se puede acceder a la aplicación a través del menú del sistema o ejecutando en un terminal jajim.sh.

Instalación a partir del binario

Para instalar el programa desde su código fuente, se debe obtener el fichero JAJIM-source-<version>.tar.gz. El primer paso es descomprimir el fichero, luego se debe entrar en el directorio creado y ejecutar el comando "make". Esto generará un fichero JAJIM.jar que se puede ejecutar haciendo click en él, o ejecutando el comando "java -jar JAJIM.jar".

Si se desea mover la aplicación, se debe asegurar que tanto el fichero JAJIM.jar como la carpeta lib, se encuentren en el mismo directorio.
