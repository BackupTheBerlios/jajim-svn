Iniciar chat privado
 Previous Top Next
Un chat privado es aquel en el que conversan dos únicos usuarios, ya que no se puede invitar a más personas a él.

Para iniciar un chat privado, hay que ir al panel de contactos y pinchar con el botón secundario del ratón en el contacto con el que se desea conversar. Eso abrirá un menú popup, en el que se da la opción de comenzar un chat privado. Está opción sólo está disponible si el contacto se encuentra conectado en este instante.


Si el usuario selecciona la opción "Chat privado", el sistema abre una nueva ventana de conversación para mantener un chat privado.


La ventana de un chat privado es diferente a la de un chat multiusuario. Como en el primero no se pueden invitar contactos, las opciones de "Invitar contacto" y "Vetar contacto" se encuentran deshabilitadas.
