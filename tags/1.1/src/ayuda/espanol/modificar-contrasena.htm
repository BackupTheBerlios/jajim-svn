Modificar contraseña
 Previous Top Next
Para modificar la contraseña es necesario que el usuario esté conectado con la cuenta al servidor. Se habilitan dos puntos en los que el usuario puede invocar la operación, el primero se encuentra en el menú Acciones del sistema y el segundo en la barra de herramientas de la ventana principal.


Cuando se selecciona la opción el sistema muestra al usuario un formulario en el que puede introducir la nueva contraseña.


Una vez rellenado el formulario correctamente, si se selecciona "Aceptar" el sistema enviará la nueva contraseña al servidor y será esta la que se utilice en la próxima conexión con la cuenta.
