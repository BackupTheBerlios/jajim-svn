Invitar contacto a chat multiusuario
 Previous Top Next
La opción de invitar un contacto sólo esta disponible si se trabaja con un chat mulitusuario. Existen dos puntos dentro de la ventana de la conversación desde donde se puede invocar esta funcionalidad. El primero se encuentra en el menú "Acciones" con el nombre de "Invitar contacto", el segundo en la barra de herramientas de la ventana.


Cuando el usuario selecciona cualquiera de las posibles entradas, el sistema muestra un formulario en el que se pueden seleccionar los contactos que se desea invitar.



Cuando el usuario selecciona al menos un contacto y pulsa el botón "Aceptar", el sistema envía una notificación a los contactos seleccionados siempre que éstos no estén vetados.
