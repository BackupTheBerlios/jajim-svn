Modificar grupo de contactos
 Previous Top Next
Para eliminar un grupo, hay que ir al panel de contactos y pinchar con el botón secundario del ratón en el grupo que se desea modificar. Eso abrirá un menú popup.


Si se selecciona la opción, el sistema muestra un formulario en el que el usuario puede introducir un nuevo nombre para el grupo.


Cuando el usuario rellena el formulario correctamente y selecciona la opción "Aceptar", el sistema modifica el nombre del grupo seleccionado.
