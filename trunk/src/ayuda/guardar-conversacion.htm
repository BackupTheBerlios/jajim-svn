Este opción se encuentra disponible en cualquier tipo de chat. Existen dos puntos dentro de la ventana de la conversación desde donde se puede invocar esta funcionalidad. El primero se encuentra en el menú "Acciones" con el nombre de "Guardar conversación", el segundo en la barra de herramientas de la ventana.


Cuando se selecciona cualquiera de las opciones el sistema muestra el siguiente formulario.


Una vez que el usuario introduce los datos de manera correcta, el sistema guarda el fichero en formato HTML.
