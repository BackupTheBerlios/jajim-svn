En el CD-ROM del proyecto se incluye un fichero de instalación para sistemas Windows y un script de instalación para sistemas tipo Linux. También se adjunta el proyecto en un archivo jar junto con sus librerías, de modo que si el usuario no quiere utilizar el instalador o su sistema no es ni Windows, ni Linux; éste pueda utilizar la aplicación.

Antes de comenzar la instalación, debe comprobar que su sistema tiene instalada la máquina virtual de java. Se recomienda la utilización de la versión 6, o superior de la misma. No se garantiza el correcto funcionamiento de la herramienta en versiones anteriores.

Instalación en Windows

Para iniciar la instalación en Windows, hay que ejecutar el instalador del programa. Durante la ejecución del mismo se ofrece al usuario la posibilidad de seleccionar la ruta de instalación.


Cuando el usuario selecciona la ruta, el instalador comienza a copiar los ficheros necesarios en el sistema. Finalmente, incluye un enlace al programa en el menú Inicio del sistema.


Instalación en Linux

Para instalar el programa en Linux es necesario disponer del fichero JAJIM_<version>.tar.gz. Primero, se debe descomprimir el mismo, después se entra dentro del directorio generado después de la descompresión. Finalmente, se ejecuta como “root “ el script instalador.sh.

Cuando finaliza el proceso de instalación, se puede acceder a la aplicación a través del menú del sistema o ejecutando en un terminal JAJIM.sh.

Instalación a partir del jar

Si no se desea realizar una instalación o se utiliza un sistema operativo diferente de los especificados, se puede utilizar la herramienta a partir de su fichero JAR. Para ello, sólo se tiene que hacer doble click en el mismo y se lanzará el sistema. Ahora bien, la carpeta de nombre "lib" siempre debe de estar en el mismo directorio que el fichero JAR, sino el sistema fallará.
