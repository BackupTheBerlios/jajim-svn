Rechazar chat privado
 Previous Top Next
El sistema muestra una ventana popup en la que informa de que una petición de chat privado ha sido recibida.


Cuando el usuario selecciona la ventana popup, se muestra la ventana de la aplicación junto con un formulario en el que el usuario puede aceptar o rechazar la petición de chat privado.


Para rechazar la petición el usuario debe seleccionar el botón "Rechazar", en ese instante el sistema envía un mensaje que confirma al contacto que el usuario ha rechazado la petición.

Además si el usuario no selecciona la ventana popup durante la visualización de la misma, el sistema rechaza el chat por defecto.
