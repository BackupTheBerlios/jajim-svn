Renombrar fichero
 Previous Top Next
Esta opción se encuentra disponible en la ventana del gestor de transferencias, concretamente en la parte superior de la pestaña de ficheros descargados.


Cuando el usuario selecciona la opción, con el fichero que quiere renombrar marcado, el sistema abre el siguiente formulario.


Si el usuario rellena el formulario adecuadamente y selecciona el botón "Aceptar", el sistema cambia el nombre del fichero al especificado.
