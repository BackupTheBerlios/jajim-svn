Iniciar chat multiusuario
 Previous Top Next
Un chat multiusuario es aquel en el que conversan varios usuarios, se pueden realizar invitaciones y se pueden vetar contactos.

Para iniciar un chat multiusuario, hay que ir al panel de contactos y pinchar con el botón secundario del ratón en el contacto con el que se desea conversar. Eso abrirá un menú popup, en el que se da la opción de comenzar un chat multiusuario. Está opción sólo está disponible si el contacto se encuentra conectado en este instante.


Cuando el usuario selecciona la opción "Chat multiusuario", el sistema muestra un formulario en el que el usuario debe introducir el nombre de la sala del chat y el alias que va a utilizar durante la conversación.


Una vez que el usuario introduce los campos y selecciona la opción "Aceptar". El sistema crea la sala, abre una ventana de chat multiusuario y envía una invitación a la sala al contacto seleccionado.


Como se puede observar en la ventana de un chat privado se puede invitar y vetar contactos.
