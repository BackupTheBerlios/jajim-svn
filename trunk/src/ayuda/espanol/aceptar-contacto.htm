Aceptar contacto
 Previous Top Next
El sistema muestra una ventana popup en la que informa de que una petición de contacto ha sido recibida.


Si el usuario selecciona la ventana popup, se muestra la herramienta y un formulario en el que el usuario puede aceptar o rechazar la petición.


Si el usuario selecciona la opción "Aceptar" el sistema muestra un formulario en el que el usuario puede introducir el alias y el grupo del nuevo contacto.


Si el usuario rellena bien el formulario y acepta el contacto, el sistema añade al nuevo contacto a la lista de contactos y envía una notificación al mismo.
