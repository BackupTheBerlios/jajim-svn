When user start application for first time, it will try to recover account file. But this file don´t exist yet, so application launch a form in which you can create or add a new count.


Add new account is convenient, because although application can work without account, user can't access to all functionality of the application.

When user add a new account correctly, application's main window is show. In this, user can access  to functionality of the application.
