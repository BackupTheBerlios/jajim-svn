Send file

This option is available in either type of chat, private or multiuser. There are two place in which user can select this Actions, in "Actions" menu and toolbar of chat.


When user selects either option, system shows a form.


When user puts into correct data and select "OK" option, system adds file to transfer manager and sends a file transfer request to selected contacts.


When contact accepts this request, transfer will begin.
