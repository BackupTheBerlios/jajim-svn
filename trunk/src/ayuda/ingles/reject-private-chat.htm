Reject private chat
 Previous Top Next
System shows a popup window in which it sasy that a private chat request has been received.


When user selects this popup window, system shows main window and a form in which user can accept or reject this request.


For reject this request user must select "Reject" button. When this event success, system sends a message that it confirms to contact this rejected.

Also, if user don't select popup window, system reject request by default.
