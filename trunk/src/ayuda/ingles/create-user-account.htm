Create new account
  
 Previous Top Next
"Create a new account" option is available in account manager. For access it, you must select option "Add account" in the account manager.


When you select this option a form is visible and user can select either create a new account in a server or add account that it exists. In this case, user must select first option.


Then system shows a form in which user must put data of the new account.

After putting data correctly and selecting "OK" option, account manager adds new account to account list.
