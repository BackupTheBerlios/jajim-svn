Windows uninstall

In the menu "JAJIM" available in "Start" menu, option Uninstall is available. When user select this option, uninstaller show follwing form.


If you select "Unistall" option, all files add to computer during installation will be deleted, furthermore it will remove install directory and menu "JAJIM".

Linux uninstall

For uninstall application in Linux, you have to execute as root the following command:

sh /usr/share/JIM/desintalador.sh


Binary uninstall

For uninstall binary, you must delete from the system, JAJIM.jar and lib folder.
