Invite contact to multiuser chat
 Previous Top Next
Option "Invite contact" is only available in multiuser chat. There are two place in which user can select this Actions, in "Actions" menu and toolbar of multiuser chat.


When user selects either option, system shows a form in which he can select contact that he want to invite.



When user selects one or more contacts and click in "OK" button, system sends a notification to this contacts if they don´t be banner.
