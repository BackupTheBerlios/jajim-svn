Installation

Before of install, you must check that your system have Java Virtual Machine installed, version 6 or superior is recommended. Early versions can provoke that application crash.

Windows installation

For install application in Windows, user have to execute the installer. During execution of installer, user can select installation directory.


When user select installation directory, installer begin to copy required files in the system. Finally, it include a link to program in Window's menu.


Linux installation

For install application in Linux, user must have file JAJIM-<version>.tar.gz. First, you must untar the file, later enter in the directory generated in that process. Finally, you must execute script "instalador.sh" as root.

When installation process finalize, use can access to application by link in the system menu or by execute jajim.sh in terminal.

Installation from binary source


For install application from binary source, user must obtain file JAJIM-source-<version>.tar.gz. First, you have to untar file and enter in generated directory. After, you must execute command "make", this operation create a file JAJIM.jar that you can execute clicked it or execute command "java -jar JAJIM.jar".

If you want move the application, you must move JAJIM.jar and directory "lib" to same directory.
