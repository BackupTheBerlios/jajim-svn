Connection to server
 Previous Top Next
There are two place in which  user can select this action, in "Actions" menu and toolbar of the main window. Both options will be available only if user is not connected yet.


If user selects either option, system connects to server with current account, and if all is right, system will show account roster.


If system can't recover account password, it show a form in which user can put his account password.


After user put account password, system connect to server and show account roster.
