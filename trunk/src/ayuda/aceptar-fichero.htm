Aceptar fichero
 Previous Top Next
El sistema muestra una ventana popup en la que informa de que una petición de transferencia de fichero ha sido recibida.


Cuando el usuario selecciona la ventana popup, se muestra la ventana de la aplicación junto con un formulario en el que el usuario puede aceptar o rechazar la petición de transferencia.


Si se selecciona la opción "Aceptar", el sistema muestra el siguiente formulario.


Una vez introducida la ruta y seleccionado "Aceptar", el sistema añade la transferencias a la lista de transferencias en progreso del gestor y comienza la misma.
